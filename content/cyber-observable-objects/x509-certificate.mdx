The X.509 Certificate object represents the properties of an X.509 certificate, as defined by ITU recommendation X.509 [X.509](https://www.itu.int/rec/T-REC-X.509/). An X.509 Certificate object <b>MUST</b> contain at least one object specific property (other than <b>type</b>) from this object.
